# Trustlevel-Webstack

Contains trustlevel components linked together using [yarn workspaces](https://classic.yarnpkg.com/en/docs/workspaces/).

# Execution (top level)

```bash
# install and link all dependencies
yarn install

# prettify/lint all projects
yarn run fix
```
